# image_analyzer

ディレクトリまたは単体ファイルから画像を読み込み、統計・ヒストグラム・ROI・ラインプロファイル・2枚比較・品質チェック・FFT・バッチ出力を行う GUI 画像解析ツールです。

**Version: 1.0**  
**推奨OS: Windows**

---

## 起動方法

### ダブルクリック起動（推奨）

`main.py` と同じフォルダにある **`startup_image_analyzer.vbs`** をダブルクリックしてください。

### コマンドラインから起動

```bash
pip install -r requirements.txt
python main.py
```

起動時にパスを指定できます（ファイルまたはフォルダ）。

```bash
python main.py C:\path\to\images
python main.py C:\path\to\image.png
```

---

## 対応ファイル形式

| 形式 | 説明 |
|------|------|
| `.npy` | NumPy 配列ファイル |
| `.png` / `.jpg` / `.jpeg` | 一般画像 |
| `.raw` | RAW バイナリ（読み込み時に幅・高さ等を指定） |
| `.bmp` / `.tif` / `.tiff` / `.webp` | その他ラスター画像 |

## 画面構成

```
┌──────────────┬────────────────────────┬──────────────────┐
│ ファイル一覧 │      プレビュー        │  解析タブ        │
```

ファイル一覧は `image_viewer` と同じ階層ツリーです（`D0001`=フォルダ、`0001`=ファイル）。

## 機能一覧

### 1. 拡張統計

全体の min / max / mean / std / median / パーセンタイル / ゼロ率 / 飽和率 / NaN・Inf 数を左パネルに表示します。

### 2. ヒストグラム

- gray / RGB チャンネル
- bins 指定
- 累積分布（CDF）オーバーレイ
- ROI 指定時は ROI 内のみ集計
- 画像として保存可能

### 3. ROI 解析

描画ツール「ROI」で次を指定できます。

| 形状 | 操作 |
|------|------|
| 矩形 | ドラッグで範囲指定 |
| 円 | 中心から半径方向にドラッグ |
| 自由選択 | ドラッグで多角形を描画 |

ROI 内の統計を「ROI」タブに表示します。

### 4. ラインプロファイル

描画ツール「ライン」で始点→終点をドラッグすると、輝度プロファイルを表示します。

### 5. 2枚比較

比較画像を開き、MSE / MAE / PSNR / SSIM と絶対差分ヒートマップを表示します。  
上部「表示」を **差分ヒートマップ** にするとプレビューでも確認できます。

### 6. 品質チェック

- ぼけ（Laplacian 分散）
- 露出不足 / 過多
- ノイズ推定
- 解像度チェック

### 7. カラーマップ（疑似カラー）

表示モード「カラーマップ」と CMAP（viridis / jet / hot など）で単チャンネルを疑似カラー表示します。

### 8. FFT

振幅スペクトル（log）を計算・表示・保存します。表示モード「FFT」でもプレビューできます。

### 9. バッチ解析

ファイル一覧すべてに対して統計（＋品質）を算出し、**CSV** または **JSON** に一括出力します。

---

## 基本的な使い方

1. **ファイルを開く** または **フォルダを開く**
2. 左の一覧で画像を選択
3. 必要に応じて描画ツールで ROI / ラインを指定
4. 右タブでヒストグラム・品質・FFT などを確認
5. 比較したい場合は「比較画像を開く」
6. 大量処理は「バッチ」タブから CSV/JSON 出力
