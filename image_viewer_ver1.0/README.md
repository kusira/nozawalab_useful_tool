# image_viewer

ディレクトリまたは単体ファイルから画像を読み込み、プレビュー・画像処理・エクスポートを行う GUI 画像ビューワーです。

**Version: 1.0**
**推奨OS: Windows**

---

## 起動方法

### ダブルクリック起動（推奨）

`main.py` と同じフォルダにある **`startup_image_viewer.vbs`** をダブルクリックしてください。  
コマンドプロンプトを開かずに GUI が起動します。

### コマンドラインから起動

```bash
pip install -r requirements.txt
python main.py
```

プロジェクトルート（`main.py` があるディレクトリ）で実行してください。

起動時にパスを指定することもできます（ファイルまたはフォルダ）。

```bash
python main.py C:\path\to\images
python main.py C:\path\to\image.png
```

## 対応ファイル形式

| 形式 | 説明 |
|------|------|
| `.npy` | NumPy 配列ファイル |
| `.png` | PNG 画像 |
| `.jpg` / `.jpeg` | JPEG 画像 |
| `.raw` | RAW バイナリ（読み込み時に幅・高さ・チャンネル・データ型などを指定） |
| `.bmp` / `.tif` / `.tiff` / `.webp` | その他ラスター画像 |

## 画面構成

```
┌──────────────┬────────────────────────┬──────────────────┐
│ ファイル一覧 │      プレビュー        │  画像パラメータ  │
│ メタ情報     │  （ズーム・パン対応）  │  ルーペ          │
│              │                        │  エクスポート    │
└──────────────┴────────────────────────┴──────────────────┘
│ ツールバー / ステータスバー                              │
└──────────────────────────────────────────────────────────┘
```

### 左パネル：ファイル一覧

- フォルダを開くと **サブフォルダを含む階層構造** で表示されます
- フォルダ行の ▶ / ▼ で **開閉** できます
- 番号はフォルダとファイルで別体系です

| 表示 | 意味 | 色 |
|------|------|-----|
| `D0001  folder/` | フォルダ番号 | 青 |
| `0001  image.png` | ファイル番号 | 緑 |

- **ファイル番号** で前後移動・ジャンプができます（フォルダ番号では移動しません）
- 下部の **メタ情報** に shape / dtype / min / max / mean などを表示します

### 中央パネル：プレビュー

- 画像処理を適用した結果を表示します
- マウスホバーで座標・ピクセル値をステータスバーに表示します
- 下部ボタンで **左右反転・上下反転・90°回転** ができます（元画像に対する操作）

### 右パネル：画像パラメータ / ルーペ / エクスポート

スライダー変更はリアルタイムでプレビューに反映されます。

## 基本的な使い方

### 1. 画像を開く

| 操作 | 説明 |
|------|------|
| **ファイルを開く** | 1 枚または複数の画像ファイルを選択 |
| **フォルダを開く** | フォルダ内の画像を再帰的に一覧表示 |
| **追加** | 現在の一覧にファイルを追加 |

RAW ファイルを開くときは、幅・高さ・チャンネル数・オフセット・データ型・バイトオーダーを入力するダイアログが表示されます。

### 2. 画像を切り替える

| 操作 | 説明 |
|------|------|
| ◀ / ▶ | 前 / 次の画像 |
| 🎲 | ランダムに移動 |
| **移動モード** | `順番` / `ランダム` / `番号指定` |
| **番号 + 移動** | ファイル番号（例: `0005` または `5`）でジャンプ |

一覧のファイルをクリック、またはダブルクリックしても切り替えられます。

### 3. 表示操作（プレビュー）

| 操作 | 説明 |
|------|------|
| マウスホイール | ズームイン / アウト |
| 中クリック / 右ドラッグ | パン（移動） |
| Ctrl + 左ドラッグ | パン（移動） |
| **Fit** | ウィンドウに合わせて表示 |
| **100%** | 実サイズ（100%）表示 |

### 4. 画像パラメータ

右パネル「**画像パラメータ**」のスライダーで調整します。  
**リセット** ボタンですべて初期値に戻ります。

| パラメータ | 説明 | 初期値 |
|------------|------|--------|
| リサイズ | 元画像サイズに対する倍率（0.1x 〜 1.0x） | 1.0x |
| 手法 | リサイズ補間方法（NEAREST / BILINEAR / BICUBIC / LANCZOS / BOX / HAMMING） | LANCZOS（高品質） |
| 明るさ | 明るさ補正 | 100 |
| コントラスト | コントラスト補正 | 100 |
| ガンマ | ガンマ補正 | 100 |
| クリップ下限 / 上限 | 輝度のクリップ範囲 | 0 / 100 |
| ぼかし | ガウシアンぼかし | 0 |
| シャープ | アンシャープマスク | 0 |
| 二値化 | 閾値二値化（0 で無効） | 0 |
| 均等化 | ヒストグラム均等化の強度 | 0 |
| 反転 | 色反転の強度 | 0 |
| 回転 | 回転角度（°） | 0 |

リサイズは他の処理より先に適用されます。エクスポート時も処理後の画像が保存されます。

### 5. ルーペ

マウスを画像上に乗せると、カーソル付近を拡大表示します。

| 設定 | 説明 | 初期値 |
|------|------|--------|
| 有効 | ルーペの ON / OFF | ON |
| 取得半径 | 拡大する範囲（px） | 64 |
| 拡大率 | ルーペ内の拡大倍率 | 2x |

### 6. エクスポート

処理後の画像を保存します。

| ボタン | 形式 |
|--------|------|
| PNG | `.png` |
| JPEG | `.jpg` |
| NPY | `.npy`（NumPy 配列） |

- **デフォルトの保存先**: ユーザーの `Downloads` フォルダ
- ファイル名: `{元ファイル名}_processed.{拡張子}`（同名がある場合は `_1`, `_2` …）
- **保存先 → 変更** でフォルダを指定できます
- **名前を付けて保存...** で任意のパス・形式を選べます

## ショートカットキー

| キー | 操作 |
|------|------|
| `Ctrl+O` | ファイルを開く |
| `Ctrl+Shift+O` | フォルダを開く |
| `←` / `→` | 前 / 次の画像 |
| `R` | ランダム移動 |
| `Home` / `End` | 先頭 / 末尾の画像 |
| `F` | ウィンドウに合わせる |
| `+` / `-` | 拡大 / 縮小 |
| `0` | 実サイズ（100%）表示 |
| `Esc` | ルーペを閉じる |

メニュー「**ヘルプ → ショートカット一覧**」からも確認できます。

## トラブルシューティング

### Python が見つからない（VBS 起動時）

Python 3.10 以上をインストールし、PATH に追加してください。  
またはコマンドラインから `python main.py` で起動してください。

### RAW が正しく表示されない

「**RAW再読込**」から幅・高さ・チャンネル数・データ型を見直してください。  
ダイアログ下部の「想定サイズ」とファイルサイズが一致するか確認してください。

### フォルダを開いても画像が出ない

対応形式（`.npy`, `.raw`, `.png`, `.jpg` など）のファイルが含まれているか確認してください。  
サブフォルダ内の画像も再帰的に検索します。
